# discord e2ee 1.0.0

This is the first packaged release under the `discord e2ee` name.

The Windows ZIP contains the plugin source, GUI launcher, PowerShell installer, setup guide, license, and version file. The installer supports Equicord and Vencord source checkouts and can target Discord Stable, Canary, or both.

GIF picker posts are allowed. Normal HTTP and HTTPS links stay visible in the Discord message, which lets Discord build embeds for sites such as Tenor, YouTube, Imgur, Giphy, Twitch, and Streamable.

`e2ee` means end-to-end encoding in this project. The 12-digit shared number does not provide end-to-end encryption.
